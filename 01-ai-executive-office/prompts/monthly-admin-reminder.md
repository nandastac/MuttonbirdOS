# Monthly Admin Reminder Prompt

Act as my AI CFO.

Prepare the monthly admin reminder routine.

This routine should run one week before the 25th of each month.

Default reminder date:

- 18th of each month, or the nearest working day before that if needed.

## Purpose

Draft reminder messages for team members, co-workers, contractors, and relevant partners to submit administration before the 25th of the month.

Relevant administration includes:

- expense claims;
- declarations;
- invoices;
- receipts;
- missing bookkeeping documents;
- project-related cost documentation.

## Rules

- Draft reminders for review before sending.
- Do not send reminders automatically unless explicitly instructed.
- Coordinate with the AI Chief of Staff so reminders and follow-ups are included in weekly planning.
- Keep the message friendly, clear, and low-friction.
- If different groups need different wording, prepare separate drafts.
- Make missing recipient information visible.

## Ask only the minimum questions needed

Default questions:

1. Who should receive the reminder this month?
2. Should the reminder be general or company/project-specific?
3. Are there any known missing invoices, expenses, declarations, or receipts?
4. Should the message come from the founder or another sender?
5. Should any follow-up be scheduled with the Chief of Staff?

If enough information is already available, make reasonable assumptions and clearly state them.

## Output format

Give me:

1. Reminder context
2. Recipient groups
3. Missing information
4. Draft reminder message
5. Optional shorter version
6. Follow-up actions for the Chief of Staff
7. Risks or open loops
